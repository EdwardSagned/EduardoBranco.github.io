De-bloating Windows 11

If you follow this toturial, you can get more performance out of your PC for free.

Nowadays, it's widely recognized that big corporations prioritize maximizing profits, often overwhelming free users with unnecessary features. Microsoft, with its recent products Windows 10 and 11, is notorious for including bloatware that hampers system performance. The worst part is how difficult and time-consuming it can be to uninstall or disable these features. This also raises privacy concerns due to the extensive telemetry data Microsoft collects. This tutorial will guide you through a quick and easy way to configure the latest Windows to your preferences.

For this we are going to use an open-source free tool called Chris Titus Tech Tool, a tool AIO (all in one) made by a famous Youtuber in the technology space. Gains of 20% are possible after using this tool, of course this depends on your system.

Install

Step 1

Search for Power Shell on your Windows.

Right click and run it as administrator.

Accept the administrator permissions and advance to the next step

Step 2

Write the next command on the Power Shell console.

iwr -useb https://christitus.com/win | iex

Press Enter and wait. The program will automatically install and open.

Program Guide

This program consists of 5 tabs

Install

Here you can install and/or update famous aplications in a quick and easy way. It uses commands to get aplications via WinGet.

Tweaks

Here is the main part of this program, it can deactivate some programs and telemetry on Windows making you PC have more avaiable resources. There are 2 profiles that select diferent tweaks, its recomended that you use the Standard profile which selects most of the sripts to run, the Minimal profile selects only the main scripts to run, it's less intrusive and faster to run.

We used the following scripts for testing:

Config

Enables you to personalize some more advanced Windows settings, we don't recommend using this part of the program if you don't know what it means. There are also the main Windows Legacy Menus in a shortcut list.

Updates

Enables you to select one of 3 updates states. Default, Security and Disable all updates. This connects to Windows Updates Services and makes it so you recieve only part of the updates, if you want to use this, its recommended to use the Security Profile.

MicroWin

This is a usefull tool for more advanced users or companies, its used to make a personalized Windows install disk/image, it makes the process of setting up new PC's much faster in larger deployments.

Conclusion

This tool was tested on Windows 11 Pro, this system is for personal use and used for education, games, and entertainment. The PC has the following specs:

CPU: Intel i5 12400F,RAM: 16GB DDR4 2666Mhz GPU:RTX 3060 12GB

After running the Standart selection of scripts and restarting the system, we registered a gain of 3% on average frames per second on the Rainbow Six Siege benchmark tool. It's important to highlight that the less free resources your PC has, the most these tweaks will impact your system positively, gains of 20% are possible on slower, older or used machines.
